# Deskripsi 

Aplikasi ini dibangun menggunakan Decoupled Architecture (arsitektur Terpisah) yang memisahkan secara pebuh antara ```Backend``` dan ```Frontend```. Proyek yang di kembangkan ini mengambil tema sistem informasi Rental buku atau Komik Digital (E-Library), yaitu sebuah sistem yang berfungsi untuk mengelola data buku, kategori genre, data penulis atau penerbit, status peminjamaan, serta manajemen data anggota.
